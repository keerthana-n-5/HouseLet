# Task 1 - Setting Up the Development Environment

## Objective

Set up a local development environment for PHP and MySQL and configure Git & GitHub for version control.

## Tools Used

- XAMPP
- PHP
- MySQL
- Visual Studio Code
- Git
- GitHub

## Steps Performed

- Installed XAMPP.
- Started Apache and MySQL services.
- Verified the local server using http://localhost.
- Installed Visual Studio Code.
- Created the project folder.
- Installed Git.
- Initialized a Git repository.
- Created the basic project structure.
- Made the initial Git commit.
- Connected the project to GitHub.

## Project Structure

```
project/
│── index.php
│── README.md
```

## Git Commands Used

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/your-username/your-repository.git
git push -u origin main
```

## Deliverables

- Local PHP development environment configured.
- Apache and MySQL running successfully.
- Git repository initialized.
- Initial project uploaded to GitHub.
